What it watches.

OpenMetrics keeps the noisy parts compressed into one menu bar surface.

System load

CPU usage, load average, active cores, RAM, cache, wired/compressed memory, swap, disk usage and available storage.

Mac state

Battery percent and charging state, estimated time remaining, network throughput, active interface, IP address, uptime, thermal state and component temperatures.

AI limits

Claude and Codex usage, detected plan, session or weekly limits, reset windows, credits and credential status.

Control surface

Configurable menu bar labels, manual refresh, automatic refresh at 1, 2, 5 or 10 seconds, and launch at login.

Local by default.

OpenMetrics uses existing local sessions. Claude is read from Claude Code Keychain or the Claude credentials file. Codex is read from CODEX_HOME, standard config locations, or Keychain. If credentials are missing, the AI tab reports login instead of pretending.

Small enough to stay open.

No web app, no account flow, no background service stack. Just a native Swift package that turns the menu bar into a compact telemetry surface.
